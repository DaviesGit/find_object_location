## About The Project

测试结果

![result00](readme_images/result00.png)

该js代码的主要功能是帮助你找到你的目标所在的位置（路径）。如上面的例子，你查找`password`这个字符串在window对象中的哪个位置，它的返回结果表明：这个字符串在windows的obj1属性的obj2属性的abc属性的第一个元素的第二个元素的asdf属性中。他可以帮助你快速获取一个网站中有用信息的位置，帮助你更快的hack一个网站。

它有5个参数：分别是`查找的对象`（可为任意合法对象）、`在哪里查找`（一般为`window`)、`跳过的对象`（一般为空）、`比较器`、`是否包含默认属性`（一般为`false`）。

你可以通过设置`比较器`来查找任意对象，可以是`函数`、`函数名`、`属性名`、`区间`、`特征值`（正则表达式匹配）。

功能:
* 快速查找指定对象的位置。
